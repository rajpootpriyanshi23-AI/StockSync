# StockSync
A Java console application that simulates basic stock trading with portfolio management.

## Features
- Buy stocks
- Sell stocks
- Track portfolio value
- Display available balance
- Object-Oriented design

## Technologies
- Java
- OOP
- Collections

## Project Structure
```
StockSync/
│
├── StockSync.java
├── README.md

## How to Run
1. Compile the project
2. Run `StockSync.java`
3. Choose Buy, Sell or Portfolio options
